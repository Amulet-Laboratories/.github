## About

Org-level GitHub configuration for Amulet Laboratories. Contains shared community health files, CI workflow templates, issue/PR templates, and the org profile README that appears on the [GitHub organization page](https://github.com/Amulet-Laboratories).

## Contents

| File / Directory | Purpose |
|-----------------|---------|
| `profile/README.md` | Org profile displayed on GitHub |
| `CONTRIBUTING.md` | Org-wide contributing guide |
| `SECURITY.md` | Vulnerability reporting policy |
| `CODE_OF_CONDUCT.md` | Community code of conduct |
| `LICENSE` | Default MIT license |
| `FUNDING.yml` | GitHub Sponsors configuration |
| `ISSUE_TEMPLATE/` | Issue templates for all repos |
| `pull_request_template.md` | PR template for all repos |
| `workflow-templates/` | Reusable CI workflow templates |
| `README-TEMPLATE.md` | Standard README template for new repos |
| `AUDIT-PLAN.md` | Infrastructure audit plan & checklist |
| `assets/` | Shared assets (brand mark SVG, etc.) |
