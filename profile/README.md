All repos share a single VS Code workspace and unified toolchain.

1. Clone [config.amulet.ink](https://github.com/Amulet-Laboratories/config.amulet.ink) — setup scripts, conventions, and the workspace file live here
2. Follow `docs/SETUP.md` in that repo for first-time bootstrap
