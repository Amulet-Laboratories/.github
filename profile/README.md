## Private Repository Policy

All repositories in this organization are **private**. We keep our work private for:
- Client confidentiality
- Competitive advantage
- Work-in-progress protection
- Operational security

Selected projects may be open-sourced with explicit decisions (e.g., `vscodemock`).

See [repo-visibility-policy.md](../../library.amulet.ink/AmuletLabsVault/01-operations/repo-visibility-policy.md) for full guidelines.
